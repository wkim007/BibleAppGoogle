# Bible Memorize App

Starter Android app for memorizing Bible verses with a progressive word-hiding practice flow.

## What is included

- Kotlin Android app using Jetpack Compose
- Three starter verses stored locally
- Prompt card for recall cues
- Progress indicator that hides more words as you practice

## Open in Android Studio

1. Open `/Users/MacBook/Desktop/AI_Project/BibleAppGoogle`.
2. Let Android Studio sync the Gradle project.
3. Run the `app` configuration on an emulator or Android device.
